# 🎬 Cine Stream

A modern movie application built with **Next.js**, **Redux Toolkit**, **Tailwind CSS**, **Jest**, and **React Testing Library**.

---

## 🚀 Features

* 🎥 Browse Popular Movies
* 🔍 Search Movies
* ❤️ Add Movies to Favorites
* 🌙 Dark / Light Theme
* 📱 Responsive UI
* ⚡ Fast Performance using Next.js
* 🧪 Unit Testing with Jest
* 🧪 Component Testing with React Testing Library
* 🌐 API Mocking
* 📊 Test Coverage Report (85%)

---

## 🛠️ Tech Stack

* Next.js
* React
* Redux Toolkit
* Tailwind CSS
* TMDB API
* Jest
* React Testing Library

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Shubhamchaurasiya1066/cine-stream.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## 🧪 Run Tests

```bash
npm test
```

Run Coverage Report

```bash
npx jest --coverage
```
